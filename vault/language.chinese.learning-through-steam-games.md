## 간체 vs 번체

### 기본 차이점

**문자 체계만의 차이**

- 간체와 번체는 같은 중국어를 다르게 쓴 것
- 기본 문법, 문장 구조, 어순은 동일

**예시**:

```
간체: 我喜欢吃饭
번체: 我喜歡吃飯
→ 둘 다 "나는 밥 먹는 것을 좋아한다" (구조 동일)
```

### 실무에서의 차이

**어휘 차이**

- 간체: "汽车" vs 번체: "汽車" 또는 "車子"
- 간체: "电脑" vs 번체: "電腦"

**용법 차이**

- 간체는 신조어, 외래어를 직역으로 많이 만듦
- 번체는 고전 표현을 더 많이 유지
- 입말(구어)에서 단어 선택이 미묘하게 다름

### 게임 플레이 관점

**좋은 소식**:

- 게임의 기본 구조(지문, 대사, 시스템 메시지)는 어느 쪽을 배워도 충분히 이해 가능
- 간체 배우면 번체 게임도 읽을 수 있고, 번체 배우면 간체도 읽을 수 있음
- 게임은 신문이나 학술서처럼 어려운 표현을 많이 안 씀

**결론**: 어느 것을 배워도 무관하지만, **간체 추천** (더 많이 쓰이는 지역이 큼)

## 중국어 발음 지역 차이

### 중요한 사실

**중국어는 지역마다 발음이 완전히 다른 언어 수준**입니다.

### 지역별 발음 체계 비교

| 항목          | 표준 중국어 (만다린) | 광동어      | 대만 국어   |
| ------------- | -------------------- | ----------- | ----------- |
| **지역**      | 베이징/중국 본토     | 홍콩/광저우 | 타이베이    |
| **성조**      | 4개 (+ 경성)         | 6-9개       | 4개         |
| **음성 체계** | 부드러운 음가        | 딱딱한 음가 | 표준과 유사 |
| **자음**      | 적음                 | 복잡함      | 같음        |
| **난이도**    | 쉬움                 | 매우 어려움 | 쉬움        |

### 발음 예시: "你好" (안녕)

**표준 중국어**:

```
nǐ hǎo
니 하오 (발음: 비교적 쉬움)
```

**광동어**:

```
nei hou 또는 ley hou
(완전히 다르게 들림 - 매우 어려움)
```

**대만 국어**:

```
ni hao
(표준과 거의 같음 - 비교적 쉬움)
```

### 게임 플레이와의 연관성

**당신이 배워야 할 것: 표준 중국어 (만다린)**

- 스팀의 모든 중국어 게임은 **표준 중국어(간체)**로 음성 녹음됨
- 광동어나 다른 방언은 게임에 없음
- 귀곡팔황/용윤입지전의 음성도 표준 중국어(베이징 발음 기준)

**대만 게임(번체)는?**

- 글자는 번체이지만 **발음은 표준 중국어와 거의 같음**
- 성조 4개, 자음체계 동일
- 다만 약간의 발음 특징 차이 있음 (예: r 발음이 덜 명확)

**광동어는 왜 안 배워도 되나**

- 성조가 **6-9개** (표준은 4개)
- "你好"가 완전히 다르게 들림
- **스팀 게임에는 광동어 음성이 없음** (홍콩 게임 제외)
- 중국 본토가 시장이 크니까요

**결론**: **표준 중국어(만다린) 배우세요**

- 게임의 100% 호환
- 가장 배우기 쉬운 방언 (성조 4개)
- 나중에 다른 지역 방문하거나 다른 게임 할 때도 가장 광범위하게 통함

## 발음 학습 전략

### 목표

게임을 즐기면서 **동시에 발음을 익히기**

### 1단계: 발음 기초 (1-2주)

**우선 순위: 성조(tone) + 핵심 자음/모음**

#### 게임에서 자주 보는 단어들

```
打 (dǎ) - 때리다
剑 (jiàn) - 검
修炼 (xiūliàn) - 수련하다
功法 (gōngfǎ) - 무공
```

#### 발음 학습법

- **YouTube**: "Chinese Pinyin Sounds" (15분짜리 기초 영상)
- **앱**: Pleco, HelloChinese (발음 녹음 비교 기능)
- **핵심**: 성조 4개 반복 연습 (중국어의 50%)

⚠️ **성조가 중요한 이유**:

```
같은 음도 성조가 다르면 뜻이 완전히 달라짐

妈 (mā) - 엄마
麻 (má) - 대마
马 (mǎ) - 말
骂 (mà) - 욕함
```

---

### 2단계: 게임하면서 발음 익히기 (핵심)

#### 추천 방식: "선택적 학습"

**A. 게임 진행 중 자주 나오는 표현만**

귀곡팔황/용윤입지전에서 **반복되는 UI 단어들**:

```
确定 (quèdìng) - 확인
返回 (fǎnhuī) - 돌아가기
装备 (zhuāngbèi) - 장비
升级 (shēngjí) - 레벨업
任务 (rènwù) - 퀘스트
战斗 (zhàndòu) - 전투
```

이 단어들은 **게임 할 때마다 수십 번 마주치니** 자연스럽게 발음이 익혀집니다.

**B. 게임 음성으로 배우기**

- 귀곡팔황: 중국어 음성 있음
- **플레이 팁**: 음성 켜고 게임 → 대사 듣고 반복 따라 읽기

효과가 진짜 좋습니다. 게임하면서 자연스럽게 성조와 발음 귀가 뚫려요.

---

### 3단계: 체계적 병행 (추천 구성)

주 3-4시간 기준:

| 시간           | 활동                | 목표                  |
| -------------- | ------------------- | --------------------- |
| **30분**       | Pinyin 기초 영상/앱 | 성조, 자음/모음 정리  |
| **2시간**      | 게임 (음성 ON)      | 게임 진행 + 발음 노출 |
| **30분-1시간** | 게임 단어 + Anki    | 자주 나오는 단어 복습 |

---

### 4단계: 구체적 도구 추천

#### 발음 학습

1. **YouTube**: "Chinese Pronunciation Guide for Beginners" (자막 있는 것)
2. **Anki 덱**: "Chinese HSK 1-2" (게임 단어 추가하면서 사용)
3. **Forvo**: 단어 발음 검색 (원어민 음성)
4. **PlEco 앱**: 단어 검색 + 발음 듣기

#### 게임 + 발음

- 게임하면서 **모르는 단어 pinyin 검색** → Anki에 추가
- 매주 "게임에서 배운 단어 20개" 발음 연습

---

### 현실적 팁

#### ⚠️ 하지 말 것

- 게임 시작 전에 모든 발음 완벽히 배우려는 것 (너무 오래 걸림)
- 게임 중에 모든 단어 발음 확인하기 (게임 흐름 끊김)

#### ✅ 추천

- 첫 1-2주: 성조 + 기초 자음/모음만
- 2주부터: **게임 진행 우선** + 자주 나오는 단어만 별도 복습
- 3개월 정도면 게임에서 나오는 주요 표현은 귀에 들어옵니다

---

### 게임별 활용 팁

#### 귀곡팔황/용윤입지전 (간체)

- 게임 자체가 느리게 진행되는 RPG → **발음 연습하기 좋음**
- 중국어 음성 있으니 따라 읽기 가능
- NPC 이름, 무공명(기술명) 반복 노출

#### 하락군협전 (번체)

- 간체와 글자는 조금 다르지만 **발음은 거의 같음**
- 유저 한글패치가 있으니 나중에 도전해도 됨

---

### 최종 로드맵

```
1주차: 성조 집중 → Pinyin 기초 (1-2시간/일)
     ↓
2-4주차: 게임 시작 (음성 ON) + 게임 단어 30개 암기
     ↓
1-3개월: 게임 진행하면서 자연스럽게 발음 습득
```

#### 솔직한 예상

- **3개월 정도면** 게임 자주 나오는 표현은 자연스럽게 발음됨
- 게임만 하면 "읽는 것"은 빨리 되지만, "듣는 것"은 시간 걸림
- 음성 켜고 진행하면 둘 다 병행 가능
