Update WSL kernel

To update the WSL kernel to the latest version, use these steps:

  1. Open Start.
  2. Search for Command Prompt, right-click the top result, and select the Run as administrator option.
  3. Type the following command to update the WSL kernelÂ and press Enter:
wsl --update

Set default version

Type the following command to set Windows Subsystem for Linux 2 your default architecture for new distros youâ€™ll install and press Enter:

wsl --set-default-version 2

Type the following command to get a list of all the distros installed on your device and press Enter:

wsl -l -v

Type the following command to convert the distro from WSL 1 to WSL 2 and press Enter:

wsl --set-version DISTRO-NAME 2
